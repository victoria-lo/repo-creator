## Remote Github Repository Creator CLI
### Installation
```
npm install --global @victoria-lo/repo-create
```
### Usage
1. Initializes the tool
```
repo-create init
```
2. Deletes stored token
```
repo-create token
```
3. Display help
```
repo-create
```
